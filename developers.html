Developer API

Programmatic access to Quantra’s live prices, the movers radar, and the public track record — authenticated by your workspace API key.
🔑 Your API key is created automatically with your workspace. Sign in on the terminal, open the account menu — the owner sees the qk_live_… key. Keep it secret; it identifies your workspace.

Authentication

Pass your key in the X-API-Key header (recommended) or as a ?key= query parameter. Base URL: https://quantra-ai.onrender.com. Rate limit: 120 requests/minute per key.

curl -H "X-API-Key: qk_live_your_key" \
  https://quantra-ai.onrender.com/api/v1/price?symbol=AAPL

Endpoints

GET/api/v1/price
Live price for any symbol. Params: symbol (required), type (crypto/stock/etf/commodity/index/fx; default stock).
GET/api/v1/movers
The movers radar — modeled odds of ±2–40% moves across the scanned universe, 1h–30d, per asset.
GET/api/v1/track-record
The live, tamper-evident accuracy record — directional hit rate and 80%-band calibration.

Example response — /api/v1/price

{
  "ok": true, "symbol": "AAPL", "price": 318.91,

All API output is probabilistic and informational — never investment advice. Redistribution of underlying exchange data may require the relevant exchange’s licence. Higher-volume and licensed-data access ships with paid plans.
