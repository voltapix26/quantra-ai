How the Quantra Score has actually done

Every day we snapshot each asset's Quantra Score and later measure the realised forward return — reporting out-of-sample directional accuracy and 80% band calibration, honestly, with no back-filling.

Embed the live badge

Drop this on a landing page or deck. It renders the current accuracy, sample size, and as-of date straight from the server — no numbers to hand-edit, nothing to fake.

Method

For each daily snapshot, assets the score called bullish (score ≥ 56) or bearish (score < 45) are checked against the actual price move at each horizon. “Directional accuracy” is the share of those calls that were right. We also show the average forward return across all scored assets. This is a transparency metric — not a performance guarantee. Markets are uncertain; expect figures only modestly above 50%.
